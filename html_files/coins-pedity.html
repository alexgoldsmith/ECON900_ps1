About Pedity

Pedity (PEDI) describes itself as a 'decentralized content platform taking a novel approach with reward sharing'. The Pedity technology stack consists of Stellar, IPFS, Stellar smart contracts, node codes based on golang, and a Javascript frontend. The project uses PEDI tokens to reward IPFS nodes that provide decentralized content storage.

The project was launched on 7 December 2017, with the first usable version being released in February 2018 using XLM as a currency. The decentralized version of Pedity was launched on 31 October 2018. Pedity completed a token sale from 6 May - 4 June 2018. Pedity launched an Android wallet in July 2018 which was published on the Google Play store in September 2018.

The project aims to:

1. Support decentralized content sharing and unbiased rewarding of users

2. Reward nodes that store IPFS content using PEDI

3. Allow contributors to share goals with their readers

4. Give public keys an identity by using user profiles
